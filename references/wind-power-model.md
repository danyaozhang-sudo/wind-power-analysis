## 二、财务模型核心计算

### 2.1 投资计算

```
总投资 = EPC单价 × 装机容量 × 10,000（元 → 万元）
资本金 = 总投资 × 20%
贷款   = 总投资 × 80%
年发电量 = 装机容量(MW) × 利用小时数 × (1-限电率) × 1,000（kWh → 万kWh）
```

### 2.2 折旧

```
折旧基数 = 总投资 × 97%（不再扣除进项税抵扣）
年折旧   = 折旧基数 ÷ 运营年限
残值     = 总投资 × 3%（第20年末回收）
```

### 2.3 贷款（等额本金）

- 每月本金相同：`贷款额 × 10,000 / 融资期限 / 12`（元/月）
- 每月利息递减：每月末剩余本金 × 月利率
- 年利息 = 12个月利息之和
- 年本金 = 月本金 × 12

### 2.4 收入计算

```
发电收入（含税） = 年发电量 × 电价 × 1.13（万元）
发电收入（不含税）= 年发电量 × 电价（万元）
```

### 2.5 进项税抵扣

- 仅在融资期限内可抵扣
- `进项税抵扣 = (年本金 + 年利息) × 13%`（万元）

### 2.6 运营成本

```
年运维费用 = 装机容量 × 10,000 × O&M费率（元/W）→ 万元
年保险费   = 总投资 × 保险费率
年备件费   = 总投资 × 备件费率
运营成本   = 运维费 + 保险费 + 备件费
```

### 2.7 税金计算

```
销项税 = 年发电量 × 电价 × 13%（万元）
城建税及附加 = 销项税 × 7%（万元）
```

### 2.8 全投资现金流量表

```
全投资CF = 发电收入(含税) + 进项税抵扣 - 运营成本 - 销项税 - 城建税及附加 - 调整所得税
调整所得税 = max(0, 发电收入不含税 - 运营成本 - 折旧) × 25%
（注意：全投资口径不加回利息）
```

### 2.9 资本金现金流量表

```
资本金CF = 发电收入(含税) + 进项税抵扣 - 运营成本 - 销项税 - 城建税及附加 - 所得税 - 本金 - 利息
PBT(equity) = 发电收入不含税 - 运营成本 - 折旧 - 利息
所得税 = max(0, PBT(equity)) × 25%
```

### 2.10 DSCR（偿债覆盖率）

```
DSCR = (净利润 + 折旧 + 利息) / (本金 + 利息)

判断标准：
  DSCR > 1.2  → 健康
  1.0 < DSCR ≤ 1.2 → 预警
  DSCR ≤ 1.0  → 危险
  贷款还清后 → 已还清
```

### 2.11 FCFE

```
FCFE = 净利润 + 折旧 + 利息抵税(×75%) - 本金支出
```

## 三、关键指标

| 指标 | 公式 | 判断标准 |
|------|------|---------|
| 全投资IRR | IRR([-总投资] + [全投资CF序列]) | >WACC=5.5% |
| 资本金IRR | IRR([-资本金] + [资本金CF序列]) | >12% 门槛 |
| 全投资NPV | NPV(WACC=5.5%, 全投资CF序列) | >0 |
| 资本金NPV | NPV(equity_cost=8%, 资本金CF序列) | >0 |
| ROE（年均） | 年均净利润 / 资本金 | 越高越好 |
| ROI（累计） | 累计净收入 / 总投资 | 越高越好 |

## 四、敏感性分析（5项）

对以下5项变量各计算7个情景，输出对资本金IRR的影响：

1. **电价**（CFD价格）：±5%、±10%、±20%
2. **利用小时数**：同上
3. **总投资**（EPC单价变化）：±5%、±10%
4. **贷款利率**：同上
5. **运维成本**（O&M费率）：±25%、±50%、±100%
